Технологии


Figma — это облачное приложение для создания дизайнов интерфейсов и прототипов, которое позволяет работать над проектами совместно в реальном времени.

Canva — это онлайн-сервис для создания графики и презентаций с простыми шаблонами и интерфейсом. Подходит для быстрого и удобного оформления визуальных материалов.

Visual Studio Code — это редактор кода от Microsoft, предназначенный для разработки ПО. Поддерживает множество языков программирования, обладает расширяемой функциональностью и удобным интерфейсом

Visual Studio — это мощная интегрированная среда разработки, предназначенная для создания приложений на различных языках, таких как C#, C++ и другие. Она обеспечивает широкий набор инструментов для программирования, отладки и тестирования программного обеспечения.
